# Meow My Crop!（喵了个菜！）AutoFarm MOD v1.7

四项功能现在完全分开控制，并会记住开关状态。

## 独立开关

- **F5：偷菜＋被偷菜**  
  关闭后不会再自动发生偷菜或被偷菜。与 F7 独立：即使暂停自动收获，F5 开启时每盆成熟作物仍只触发一次偷菜＋被偷菜。
- **F6：自动按键输送**  
  每 4 帧自动发送一次 A 键成长消息（60 FPS 下约每秒 15 次），避免发送过密拖慢游戏。关闭后停止发送，不影响真实键盘、鼠标输入，也不会抢鼠标。
- **F7：自动种植收获**  
  关闭后停止自动收获和自动补种，仍可手动种植和收获。
- **F8：自动开罐**  
  关闭后停止自动连续开罐，手动“开罐”按钮仍可正常使用。

四项功能首次运行均默认开启，各开关互不影响。状态通过 Unity PlayerPrefs 保存，退出并重开游戏后仍会保持。

## 保留功能

- 指定作物种一次后自动收获并持续补种同一种作物。
- 1x / 2x / 5x / 10x / 20x / 50x 生长速度。
- 离线自动偷菜和自动被偷菜。
- 自动开罐。
- 装饰在本地按橙色 Legendary 品质显示和分类。

## 安装

1. 完全退出游戏。
2. 下载并解压项目文件。
3. 双击 `1_INSTALL_MOD.cmd`。
4. 安装完成后进入游戏，使用 F5～F8 分别开关四项功能。

## 重要说明

罐头奖励仍由 Steam Inventory 决定。橙色品质修改是客户端本地显示与分类覆盖，不能把 Steam 服务器库存里的普通物品真正改造成可交易的传奇物品。
